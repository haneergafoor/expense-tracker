GETTING STARTED PATHS
================================================================================

PATH 1: Direct (Fastest)
  1. Open index.html in browser
  2. Start using immediately
  Time: 10 seconds

PATH 2: HTTP Server (Better)
  1. cd to project directory
  2. Run: python3 -m http.server 8000
  3. Visit: http://localhost:8000
  Time: 30 seconds

PATH 3: Setup Script (Best)
  1. Run: ./setup.sh
  2. Follows from there (detects setup)
  Time: 1 minute

PATH 4: Flask Server (Enhanced)
  1. Ensure Flask installed
  2. Run: python3 server.py
  3. Visit: http://localhost:5000
  Time: 2 minutes

================================================================================

  - Backup JSON file to safe location
  - Test restore process quarterly

Data Protection:
  - All data stored locally (private)
  - Browser LocalStorage: ~5-10MB limit
  - Equivalent: ~50,000 transactions
  - No automatic upload (completely private)

Cleanup:
  - Archive old data if needed
  - Use JSON export for long-term storage
  - Clear cache only if needed

================================================================================
TROUBLESHOOTING REFERENCE
================================================================================

Issue: App won't open
  Solution: Try direct browser file open or HTTP server

Issue: Data disappeared
  Solution: Restore from JSON backup or check browser settings

Issue: Charts not showing
  Solution: Refresh page, ensure at least 1 expense exists

Issue: Slow performance
  Solution: Normal for 1000+ expenses, use Flask server

Issue: Settings not saving
  Solution: Refresh page after saving

More help: See README.md or START_HERE.md
