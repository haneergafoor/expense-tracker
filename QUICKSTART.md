# Quick Start Guide 🚀

## Option 1: Direct HTML (Fastest ⚡)
```bash
# Simply open in your browser:
# Right-click on index.html → Open with → Your Browser
# Or type in URL bar:
file:///home/haneer/Documents/expence\ tracker/index.html
```

## Option 2: Python HTTP Server
```bash
cd "/home/haneer/Documents/expence tracker"
python3 -m http.server 8000
# Visit: http://localhost:8000
```

## Option 3: Flask Server (Best Experience)
```bash
# Install Flask first (if not installed):
pip3 install flask

# Run:
python3 server.py
# Visit: http://localhost:5000
```

## Option 4: Using the Setup Script
```bash
chmod +x setup.sh
./setup.sh
```
